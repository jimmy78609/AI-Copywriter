🔑 如何申請 API Key

🌟 Google Gemini (推薦 - 免費)

  1. 前往 Google AI Studio
  2. 用Google帳號登入
  3. 點擊「Create API Key」
  4. 複製產生的API Key
  5. 回到這裡貼上即可使用

免費額度：每月15次免費調用

🤖 OpenAI GPT-4

  1. 前往 OpenAI Platform
  2. 註冊或登入帳號
  3. 點擊「Create new secret key」
  4. 複製產生的API Key
  5. 需要綁定信用卡，但新用戶有$5免費額度

成本：約$0.03/次

🧠 Anthropic Claude

  1. 前往 Anthropic Console
  2. 註冊帳號
  3. 點擊「Create Key」
  4. 複製API Key
  5. 需要綁定信用卡

成本：約$0.015/次，品質最高
